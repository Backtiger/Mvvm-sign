# Mvvm-sign
MVVM패턴 기반의 WPF 전자서명 프로그램
 
hwpcrtl을 사용한 한글 문서 기반의 이미지 서명 프로그램입니다.
Mvvm 디자인 패턴을 최대한 준수하며 개발에 유의하였습니다.



## 1. 개발환경
* IDE<br> 
 visualstudio2022
* framwork<br> 
.net 4.8     
*  확장 <br>  Material Design tool https://github.com/MaterialDesignInXAML/MaterialDesignInXamlToolkit
